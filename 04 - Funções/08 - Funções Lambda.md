# Funções Lambda



- Também conhecidas como **Funções Anônimas**.
- São uma forma de criar funções pequenas e temporárias em Python sem a necessidade de usar a palavra-chave ``def``.

Sintaxe:
````python
lambda argumentos: expressao
````


Exemplo:
```python
dobrar = lambda x: x * 2
print(dobrar(5))  # Saída: 10
```
