# Operadores Lógicos (Comparação)


```python
idade = 15


print(idade > 17)
print(idade < 17)
print(idade >= 17)
print(idade <= 17)
print(idade == 17)
print(idade != 17)


# Comparações entre Outros Tipos


print(True == True)  # True
print('Rafael' == 'rafael')  # False
print('b' > 'a')  # True
print(5 == '5')  # False
```
